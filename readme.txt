# Acerca del desafío

Requisitos: Poseer conocimiento de programación en html, css y js y manejo de control de versiones git.

Objetivos: Demostrar tus habilidades en desarrollo web para poder acceder al bootcamp.
Explicación del desafío: Deberán crear un proyecto  con js vanilla, html y css (sin librería ni framework), 
donde deberán consumir la siguiente api: > https://restcountries.com/. 

De cada uno de los países deberán mostrar en la primera interfaz en tarjetas la siguiente información: 
imagen de la bandera, population, región, capital y crear un vínculo que te lleve a otra interfaz.

Para ver solo el detalle de ese país y tener un botón para volver a la interfaz inicial.

A su vez, en la primera interfaz deberán tener un buscador de países y un filtro/selector que permita filtrar por región. 

Para terminar.
El proyecto debe venir en versión responsive.

Plazo de entrega: 2 semanas.
Forma de entrega: En el espacio designado para la entrega del trabajo se debe de dejar el link de un repositorio
de github con la solución que realizo para el proyecto como también otro de
github pages donde se despliegue la aplicación correctamente. 
Por lo tanto, el repositorio debe ser público y además de estar desplegado. Si no se cumplen esos dos últimos parámetros no se calificará su proyecto.
